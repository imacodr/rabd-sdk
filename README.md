## About

A wrapper for the RABD REST API. Only made for `hub.rabd.space` endpoints at the moment.

## Installation

```bash
npm install rabdapi
```

or

```bash
yarn add rabdapi
```

## Documentation

Documentation for the rabdAPI package can be found at the [RABD Developer Hub](https://developer.rabd.space/)
